# 한국어 자막 스타일 가이드

SceneWeaver가 생성하는 SRT 파일의 품질 기준. `build-subtitles` 스킬과 사람 편집자 모두 이 규칙을 따른다.

## 파일 포맷

- **인코딩**: UTF-8 **BOM 있음** (`\uFEFF`)
- **줄바꿈**: CRLF (`\r\n`)
- **확장자**: `.srt`
- **타임스탬프 형식**: `HH:MM:SS,mmm`  (콤마, 밀리초 3자리)
- **큐 구조**:
  ```
  1
  00:00:01,500 --> 00:00:04,200
  자막 텍스트 첫 줄
  자막 텍스트 둘째 줄 (선택)

  2
  00:00:04,300 --> 00:00:07,000
  ...
  ```

## 길이 규칙

| 항목 | 기준 |
|---|---|
| 한 줄 최대 글자 수 | **18자** (한글) / 36자 (영문 기준 등가) |
| 한 큐 최대 줄 수 | 2줄 |
| 한 큐 최소 지속 시간 | 1.5초 |
| 한 큐 최대 지속 시간 | 7초 |
| 큐 사이 최소 간격 | 0.1초 (100ms) |
| 한 큐 최대 글자 수 | 36자 (2줄 × 18자) |

범위를 벗어나면 `build-subtitles` 스킬이 **편집 권장 목록**에 넣는다.

## 줄바꿈 규칙

### 반드시 지키기

- **조사 앞에서 줄바꿈 금지** — 은/는/이/가/을/를/에/에서/과/와/도/만 등
  - ✗ `심리학 교수\n스키너는 타자기 앞에`
  - ✓ `심리학 교수 스키너는\n타자기 앞에 앉아`

- **어미/접속사 뒤에서 줄바꿈 가능** — "그러나", "하지만", "그래서", "~하고", "~면서"
  - ✓ `실험은 성공했지만\n결과는 예상과 달랐다`

- **고유명사 쪼개지 않기**
  - ✗ `하버드\n대학교` → ✓ `하버드 대학교` (한 줄)

### 권장

- 의미 단위로 자르기 (주어+서술부, 수식어+피수식어)
- 2줄이 될 때 윗줄이 아랫줄보다 짧거나 같게 (시각적 안정감)

## 타이밍 규칙

### 기본 산식

```
cue_duration = (cue_chars / scene_chars) × narration_seconds
```

씬의 `narration_seconds` 를 문장별 글자수 비율로 배분. 합계는 `narration_seconds` 와 오차 ±0.1초 이내.

### 조정

- 너무 짧은 큐 (< 1.5초): 앞/뒤 큐와 **병합** (문법적으로 자연스러울 때)
- 너무 긴 큐 (> 7초): **분할** — 문장 경계, 접속사 경계에서
- 침묵 구간: 내레이션에 쉼표/문단 구분이 있으면 큐 사이 간격을 0.3~0.5초로 늘려도 됨

### 씬 간 경계

- 씬 N의 마지막 큐와 씬 N+1의 첫 큐 사이는 **최소 0.3초** (화면 전환 시간 확보)
- 씬 1은 `00:00:00,000` 또는 video_meta의 `opening_title` 길이 이후 시작

## 합본 SRT (ch{NN}_full.srt)

씬별 SRT들을 타임라인 전체로 합친 것. 큐 번호는 전역으로 1부터 재부여.

```
workspace/ch{NN}/subtitles/
├── ch{NN}_01.srt   (씬 1 로컬 타임: 0~25초)
├── ch{NN}_02.srt   (씬 2 로컬 타임: 0~20초)
├── ...
└── ch{NN}_full.srt (전역 타임: 0~120초)
```

씬별 파일은 편집 편의용, 합본은 CapCut 로드용.

## CapCut에서 잘 안 보이는 경우 체크리스트

1. 인코딩이 UTF-8 **BOM** 인가? (BOM 없으면 깨지는 경우 있음)
2. 줄바꿈이 **CRLF** 인가? (LF만 있으면 일부 CapCut 버전에서 큐 분리 실패)
3. 타임스탬프가 `HH:MM:SS,mmm` (콤마) 인가? (점은 WebVTT 포맷, CapCut은 SRT)
4. 큐 사이 빈 줄이 **한 줄씩** 있는가?
5. 큐 번호가 1부터 순차인가?

## 편집 시 주의

- SRT를 편집할 때 **줄 수가 바뀌면** 타임스탬프는 그대로 두고 텍스트만 바꾼다
- 한 큐를 둘로 나눌 때는 중간 시각을 문장 비율로 계산
- 전체 길이는 씬 `narration_seconds` 합계를 **넘지 않아야** 함 (넘으면 오디오와 어긋남)
